# 競馬倶楽部

競馬予想Webアプリの画面付きプロトタイプです。Python標準ライブラリだけで動き、SQLiteに2026年6月7日の重賞レース・出走馬・購入履歴を保存します。

## 起動

```powershell
python app.py
```

ブラウザで以下を開きます。

```text
http://127.0.0.1:8000
```

## 現在できること

- レース一覧の表示
- レース詳細の表示
- 出走馬ごとの予想スコア、勝率、複勝率、期待値の表示
- 回収率レポートの表示

現在の掲載レースは東京11R「安田記念（GⅠ）」のみです。
